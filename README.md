# STAT 184 HW 4.3 Repository

This repository contains materials for HW #4.3 for STAT 184. The project focuses on reproducible data analysis, including an airport passenger traffic analysis and a Monte Carlo numerical integration example.

## Project Purpose
This repo was created to document the work for HW #4.3 and to demonstrate the use of GitHub version control features, including branching, issues, pull requests, and commits.

## Data Source
The airport traffic values and Monte Carlo analysis are based on materials revisited from HW #3.4. The rendered PDF in this repository contains the written analysis, figures, and code appendix used for the assignment.

## Current Plan
The current plan for this project is to:
1. Maintain a public and organized GitHub repository.
2. Include the required assignment files.
3. Track progress with GitHub Issues.
4. Use branches and pull requests to manage changes.
5. Keep the main branch updated with completed work.

## Repository Organization
This repository currently includes:
- `README.md` — overview of the repository
- `plan.md` — plan for maintaining the repository and completing HW #4.3
- `HW4.3_rendered.pdf` — submitted PDF for the assignment
- `HW4.3_report.qmd` — QMD source file used for the assignment
